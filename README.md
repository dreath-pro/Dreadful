Dreadful

About
An eldritch horror turn based game, where you can choose which anomalies to fight with each other, 
this game features a simple turn base combat mechanics with a grotesque supernatural art made by AI

Game Mechanics
Select which monster you would want to fight and just watch with its battle logs to narrate what happens in a fight, 
and a dialogue for each monster during an attack or tanking damage

Tech Stack
*IDE: Android Studio
*Language: Java
*Art: Bing Image Creator
